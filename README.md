# Sistem Rekomendasi E-Commerce Menggunakan 3 Algoritma Similarity

Project ini merupakan implementasi sistem rekomendasi e-commerce menggunakan tiga algoritma similarity, yaitu:

1. Cosine Similarity
2. Pearson Correlation Coefficient
3. Jaccard Similarity

Dataset berisi data rating produk elektronik oleh pengguna dan digunakan untuk melakukan analisis kemiripan antar user maupun antar produk.
